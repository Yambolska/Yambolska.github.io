Credit Risk & Loan Default Modeling — Lending Club Dataset

A data-driven financial risk modeling case using real loan applicants.

Project Overview

This project uses real Lending Club loan data to build a machine learning model that predicts whether a loan will be fully paid or defaulted. The analysis focuses on how borrower characteristics, credit history and financial indicators correlate with repayment risk, and how clean feature engineering and model selection can improve predictive performance.

Data Preparation & Feature Engineering

  • Missing values analyzed and imputed or strategically dropped
  • High-cardinality and redundant categorical features removed
  • Created a binary target loan_paid from loan_status
  • One-hot encoded categorical variables
  • Identified correlated features and reduced multicollinearity
  • Adjusted for class imbalance and focused on precision/recall and F1-score metrics

Exploratory Insights

  • Loan grade strongly correlates with default likelihood
  • Lower grades (F and G) show extremely high default ratios
  • Loan amount and installment are nearly perfectly correlated
  • Most features provide independent predictive signals
  • Employment length and job title showed weak predictive power and were dropped

Imbalance & Evaluation Approach

Because “Fully Paid” loans significantly outnumber “Charged Off” loans, accuracy alone would be misleading. Therefore evaluation emphasized:

  • Recall for identifying defaults
  • Precision of default predictions
  • F1-score as the balanced measure

Models Used

Several machine learning models were implemented and compared:

  • Random Forest — AUC: 0.89
  • Keras Neural Network — AUC: 0.90
  • XGBoost — AUC: 0.91

All models demonstrated strong predictive capability with XGBoost performing the best overall, consistent with its strength on tabular datasets. Tree-based models also allow for feature importance analysis, improving interpretability for financial risk teams.

Interpretation & Business Meaning

  • All tested models effectively differentiate between fully-paid and charged-off loans
  • XGBoost is slightly superior and the most reliable model for production use
  • Tree-based models are highly interpretable for financial decision-support systems
  • Deep learning performs well, but does not significantly outperform boosted trees in this context

Conclusion & Potential Applications

This project shows how real consumer loan application data can be transformed into actionable default-risk predictions using structured feature engineering and machine learning. XGBoost offers the best balance of performance and interpretability, making it suitable for credit scoring applications. With further tuning and potential ensembling, the predictive framework could support underwriting decisions, reduce default exposure, and strengthen credit portfolio risk management.
